A modular Web Application Firewall that inspects HTTP requests in real time and blocks common web attacks (SQLi, XSS, Path Traversal). Includes rate limiting, brute-force protection, and a monitoring dashboard.
**Tech:** Python, Flask, Regex, SQLite
🔗 https://github.com/rizu-SM/WAF
